# Omni Apps Scripts

A collection of [Google Apps Scripts](https://developers.google.com/apps-script) for working with [Omni Analytics](https://omni.co) from inside Google Workspace.

Each subdirectory is a self-contained script with its own setup README.

## Scripts

| Directory | What it does |
|-----------|--------------|
| [`gsheets-sync/`](./gsheets-sync) | Pull data from specific Omni dashboard tiles into specific cells of a Google Sheet. Sheet-driven config, on-demand or scheduled refresh. |

## Contributing

PRs welcome. Each script lives in its own subdirectory with a `README.md` covering setup, config, and how it works. Add a row to the table above when you contribute a new one.
